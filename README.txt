Webscraping on Hepsiburada and GittiGidiyor.
This code collects the data from the given inputs, exports them into json, csv 
and excel then sends an alert email when the scraping is done.
While giving inputs, make sure you put no space after the commas.
You can change the proxy by entering the proxy that you want to use in the format of
'1.1.1.1:1111'
You will see on the code where to integrate your proxy
Make sure you are using your own headers while compiling the code.
Make sure pandas, BeautifulSoup is installed on your pc.
After running the code, it will create the exported data format items in the directory
that it is located on.
